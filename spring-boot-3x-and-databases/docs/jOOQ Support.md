# jOOQ Support

(as of 19.3)

Note: there is no support for features added in MySQL 9 or PostgreSQL 16.

## Community Edition

| Name         | SQLDialect Family   |
|--------------|---------------------|
| MariaDB      | MARIADB             |
| MySQL        | MYSQL               |
| PostgreSQL   | POSTGRES            |
| Trino        | TRINO               |
| YugabyteDB   | YUGABYTE            |

We can use generic support for the remaining databases.
